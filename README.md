## IndigoLab test project

### Requirements
- PHP >= 8.4
- Composer >= 2
- Make >= 4
- Docker

### Installation
> `git clone git@github.com:NONstop5/indigoLab-test.git`
> 
> `make install`

### Run
> `make start`
